Introduction

The CDC provides vital statistics datasets for births within the United States. This data can be found at https://www.cdc.gov/nchs/data_access/vitalstatsonline.htm. The following analysis will be reviewing the 2017 data. Within this analysis we will be looking for relations between attributes recorded about the mother and the newborn. Are there details within the dataset that show actions which the mother can take that may impact the newborn?
